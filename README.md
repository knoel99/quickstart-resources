# Exercice 1

Tutoriel officel d'Anthropic. Se créer un compte et suivre le cours qui dure ~50 min. Vous aurez une certification.

* Lien du cours: https://anthropic.skilljar.com/introduction-to-model-context-protocol
* Exemple de certification : https://verify.skilljar.com/c/t74efcv7bnn3
* `exercice1/cli_project` est le projet à initial à complété
* `exercice1/cli_project_COMPLETE` est le projet à l'état final après complétion du tutoriel

Exercice bonus : 
* Implémenter les outils MCP pour aller chercher les informations concernant votre compte GitHub comme mentionné dans le cours
